# unifacef-tcc

## Requisitos
* Java 8
* Maven

## Instalar as dependências e compilar o projeto
```sh
mvn clean install
```

## Inicializar a aplicação
```sh
mvn spring-boot:run
```

## Executar os testes
```sh
mvn clean test
```
OBS: A cobertura dos testes fica dentro [deste arquivo](./target/jacoco/index.html).
